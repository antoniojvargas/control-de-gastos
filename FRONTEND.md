# Frontend — React

## Mi nivel
Intermedio-bajo en React. Prefiero que el código sea simple y explicado.

## Stack frontend
- React 18 + Vite
- React Router v6
- Axios para llamadas API
- Tailwind CSS para estilos

## Convenciones
- Componentes funcionales siempre, nunca class components
- Un componente por archivo
- Hooks personalizados en src/hooks/
- No usar Redux, preferir Context o estado local

## Patrones preferidos
- Fetch de datos en el componente padre, pasar por props
- Loading/error states explícitos en cada componente
- Forms con React Hook Form

## Lo que Claude debe hacer
- Explicarme el "por qué" cuando haga algo no obvio en React
- Mantener componentes pequeños y legibles
- Avisarme si hay una forma más simple de hacer algo
